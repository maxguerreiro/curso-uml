# Curso UML – Modelagem de Domínio

Projeto desenvolvido com foco em **modelagem de domínio utilizando UML**, aplicando conceitos de orientação a objetos para representar entidades, relacionamentos e regras de negócio.

O objetivo deste projeto é praticar a **análise e modelagem de sistemas antes da implementação**, seguindo boas práticas de design de software.

## Conceitos aplicados

- Modelagem de classes
- Relacionamentos entre entidades
- Associações, agregações e composições
- Princípios de orientação a objetos
- Análise de domínio

## Tecnologias e ferramentas

- Java
- UML
- IDE IntelliJ / Eclipse
- Diagramas de classes

## Objetivo do projeto

Desenvolver a capacidade de **pensar o sistema antes de codificar**, facilitando a manutenção, evolução e clareza do software.

Este projeto faz parte do meu processo de formação como desenvolvedor backend.
